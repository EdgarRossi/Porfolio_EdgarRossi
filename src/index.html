PROYECTOS

E-commerce HenryFilms

Este proyecto consiste en una aplicación que permite visualizar un listado de películas y series con información detallada de cada una al darle click en este.
El proyecto cuenta con un buscador autocompletado, ordenamientos alfabéticos, ordenamientos por puntuación y filtrados por géneros.
Tiene un sistema de autenticación con login, register y login con google.
Cuenta con diferentes roles, como guest, user y admin, cada uno con mas complegidad en la funcionalidad de la pagina.
También cuenta con un carrito para agregar los productos deseados y con posibilidad de pagarlos con tarjeta de crédito, después de la compra se puede dejar una reseña en dicho articulo.
Se puede además agregar a favoritos y desde el perfil de usuario poder verlos, como también desde este perfil se pueden modificar algunos datos del usuario.

Pokemons App

Este proyecto es una Single Page Application que permite obtener información desde una API.
Van a encontrar un listado de pokemones con una información detallada de cada uno al hacerle click a la tarjeta.
Pueden crear sus propios Pokemones por medio de un formulario controlado.
También pueden filtrarlos por tipos,creados o existentes, ordenarlos alfabéticamente y por ataque.
Cuenta con un paginado con 12 pokemons por pagina.
Las herramientas utilizadas fueron: JavaScript, HTML, CSS, NodeJS, Express, Sequelize, PostgreSQL, React y Redux.
